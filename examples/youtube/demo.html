What's actually enforced vs. best-effort?

■ Hard: Scope function runs server-side, AST-validated. Query agent literally cannot see unfiltered rows.
■ Hard: Each agent runs in a separate Docker container. Bridge-only egress. Cannot reach Postgres or internet directly.
■ Hard: Budget enforced server-side. 4/50 calls used. Agent cannot exceed 200K tokens.
■ Soft: Scope function quality depends on LLM. Here it wrote a reasonable k-anonymity + redaction policy.
■ Soft: Mediator PII check is LLM-based. Could miss subtle leaks.
■ Not enforced: On-chain deploy verification (skipped for dev). No AppAuth contract deployed.
